Query ข้อมูล Order by ไม่ตรง

วิธีตรวจสอบ/แก้ไข
- ตรวจสอบ Query 
- ลอง Query Table อื่นที่มีข้อมูลใกล้เคียงกัน
- ตรวจสอบโครงสร้าง Table เทียบกับ table อื่น และ ตรวจสอบ Type ของ Column

สรุป
- Type ของ Column ของ Table ที่มีปัญหา ไม่ตรงกับของ Table อื่นที่เก็บข้อมูลใกล้เคียงกัน
- แก้ไขแบบที่ไม่กระทบโครงสร้าง Table โดยการ ใส่ฟังก์ชั่นเปลี่ยนประเภทตัวแปร ( CAST(ชื่อ Column AS SIGNED) ) ลงไปใน Query แล้ว Order by จะตรง
